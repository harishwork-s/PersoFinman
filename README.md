# PersoFinman

A mobile-first financial control system that helps users track bills, subscriptions, warranties, and payments in one place to prevent money leakage and missed obligations.

## 🚀 Features

- 📊 Dashboard with financial overview
- 💳 Autopay subscription tracking
- 🧾 Bill reminders with due alerts
- 📦 Warranty tracking with invoice upload
- ✅ Financial to-do task management
- 🌐 English / Tamil language support
- 🔗 Direct payment links

## 🛠 Tech Stack

- React Native (Expo)
- Python Flask
- SQLite
